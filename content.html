Typography Style Guide

We use a base font size of 16px on a line height of 24px (1.5) with a Modular Scale of 1.214.

This, however, is only used for the largest screens (1100px and above). On anything below 1100px, we will use a base font size of 14px. Since we use rems for sizing, all font-sizes will scale accordingly.

The Photoshop mockups use the base font-size of 14px.
